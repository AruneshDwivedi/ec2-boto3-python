# AWS EC2 Creation using Python (boto3)

This project demonstrates how to create and manage an AWS EC2 instance using Python and the boto3 library.

It automates infrastructure provisioning instead of creating resources manually from the AWS console.

---

## Objective

* Launch EC2 instance using Python
* Understand boto3 and AWS APIs
* Learn basic cloud automation

---

## Technologies Used

* Python
* boto3
* AWS EC2
* IAM
* SSH
* WSL/Linux

---

## Prerequisites

* AWS account
* IAM user with EC2 permissions
* AWS CLI configured

```bash
aws configure
```

* Python installed

```bash
pip install boto3
```

---

## Project Workflow

1. Connect Python to AWS using boto3
2. Launch EC2 instance using configuration
3. Wait until instance is running
4. Fetch public and private IP
5. Print instance details

---
## Output

* New EC2 instance created
* Instance ID displayed
* Public and Private IP printed

## Key Concepts

* boto3 SDK
* EC2 provisioning
* IAM authentication
* Infrastructure as Code
